# ChargeGo – Smart Powerbank Sharing Network

ChargeGo is a smart powerbank-sharing platform designed for travelers and commuters facing low-battery emergencies. Users can rent charged powerbanks from nearby kiosks and return them at any partner station. The project solves the problem of unavailable public charging outlets during travel, shopping, or commuting. It includes features like station finder, QR rental system, wallet payments, subscriptions, and admin dashboard. The UI is designed in Figma with a premium modern startup look. ChargeGo aims to make mobile charging accessible anytime, anywhere.
